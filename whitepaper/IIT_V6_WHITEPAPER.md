## 5. Φ-Preserving Quantum Error Correction

### 5.1 The Φ–QEC Compatibility Problem

Standard QEC syndromes destroy quantum coherence in ancilla registers during error detection, potentially collapsing the Φ substrate. IIT v6.0 introduces **Φ-preserving QEC** (Φ-QEC), a protocol that interleaves error correction with ancilla-mediated Φ readout such that the Φ measurement and QEC syndrome extraction commute.

### 5.2 Φ-QEC Protocol

```
1. Encode logical state |ψ_L⟩ in distance-d surface code
2. Prepare ancilla registers A_Φ (Φ readout) and A_synd (syndrome extraction)
3. Apply: CX(|ψ_L⟩, A_Φ) — copy Φ-relevant correlations to A_Φ ancilla
4. Apply: stabilizer measurements on A_synd (does NOT touch A_Φ)
5. Apply corrections to |ψ_L⟩ based on A_synd
6. Measure A_Φ → Φ value extracted
7. Verify: [CX_Φ, S_{synd}] = 0 (commutativity check)
```

The commutativity condition in step 7 is guaranteed when the Φ ancilla CX gates are applied only to logical qubits and the syndrome ancillas are in a separate register.

### 5.3 Φ-QEC Fidelity Bound

**Theorem (Φ-QEC Fidelity)**: Under Φ-QEC with code distance d and per-qubit physical error rate p < p_th:

```
F_Φ-QEC ≥ 1 - O((p/p_th)^{⌈(d+1)/2⌉})
```

For d = 7 and p/p_th = 0.1, this gives F_Φ-QEC ≥ 1 - 10⁻⁴, exceeding the v5.0 ancilla fidelity of 97.8%.
